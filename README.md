# SkillSync AI

AI-powered freelancer marketplace with a handcrafted, premium dark UI.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing with interactive hero |
| `/talent` | Freelancer discovery |
| `/profile/:id` | Immersive profile pages |
| `/portfolio` | Masonry portfolio gallery |
| `/projects` | Project list |
| `/projects/vault-redesign` | Notion-style project doc |
| `/dashboard` | 3-column workspace + AI assistant |
| `/community` | LinkedIn/Reddit/Discord hybrid |
| `/mentorship` | Mentor matching |
| `/ai/smart-match` | Compatibility visualization |
| `/ai/proposal-evaluator` | Animated score breakdown |
| `/ai/skill-verification` | Interactive quiz flow |

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS v4
- Framer Motion
- React Router
